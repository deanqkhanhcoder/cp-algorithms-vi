# Contributing

This repository is a Vietnamese translation of the original
cp-algorithms project.

## How to contribute

You can contribute by:
- Fixing translation mistakes
- Improving clarity or terminology
- Translating new articles

Please note:
- Algorithm logic, formulas, and code should remain unchanged
- Terminology should follow common competitive programming usage

Contributions related to the original English content should be made
directly to the upstream repository:
https://github.com/cp-algorithms/cp-algorithms
