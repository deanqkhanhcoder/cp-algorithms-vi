---

## What should be reported here?

You may open an **Issue** in this repository if you find:

- Malicious or suspicious content in translated text
- Incorrect links leading to unsafe resources
- Accidental exposure of sensitive information (tokens, credentials, etc.)
- Translation errors that could cause **misunderstanding of algorithms**

---

## What should NOT be reported here?

Please **do NOT** report the following to this repository:

- Security vulnerabilities in the original cp-algorithms project  
- Issues related to GitHub Pages, MkDocs, or hosting infrastructure  
- Vulnerabilities in algorithms themselves (these are academic descriptions)
