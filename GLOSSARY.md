# CP-Algorithms Vietnam - Glossary (Thuật ngữ)

Bảng dưới đây thống nhất cách dịch các thuật ngữ kỹ thuật trong dự án. Vui lòng tuân thủ nghiêm ngặt để đảm bảo sự nhất quán trên toàn bộ tài liệu.

## Cấu trúc dữ liệu & Giải thuật cơ bản

| English Term | Vietnamese Chuẩn | Ghi chú / Ngữ cảnh |
| :--- | :--- | :--- |
| **Array** | Mảng | |
| **Subarray** | Mảng con | Một phần liên tiếp của mảng. |
| **Subsequence** | Dãy con | Không nhất thiết liên tiếp, nhưng giữ nguyên thứ tự. |
| **Substring** | Xâu con | Một phần liên tiếp của chuỗi ký tự. |
| **String** | Xâu / Chuỗi | Thống nhất dùng "Xâu" cho string trong CP. |
| **Queue** | Hàng đợi | |
| **Stack** | Ngăn xếp | |
| **Heap** | Heap / Đống | Nên giữ "Heap" hoặc dùng "Đống" (nhưng Heap phổ biến hơn). |
| **Set** | Tập hợp | |
| **Map** | Ánh xạ / Map | Thường giữ nguyên "Map" khi nói về CTDL (std::map). |

## Quy tắc chung
1. **Tên thuật toán riêng**: Giữ nguyên tiếng Anh hoặc để song ngữ lần đầu tiên xuất hiện.
   - VD: *KMP Algorithm*, *Z-function*, *Convex Hull Trick*.
2. **Từ viết tắt**: Viết hoa toàn bộ.
   - BFS, DFS, MST, DP, RMQ, LCA.
